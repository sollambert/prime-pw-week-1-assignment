# 4 - What is a quick way to create a new folder?

# 5 - What output would you expect when you run the command 'ls' in an empty folder? What about a folder containing files?
No output for an empty directory, and a list of all items contained within directory for one that is not empty.

# 8 - Why would you run the command 'git status'?
To determine how far out of sync your current working directory on your local PC is from the main branch of the project you're working on.

# 9 - What does the command 'git add .' do?
It adds the changes you've made to your local files to the next commit.

# 10 - What is the command to push your committed changes to your repo on GitHub?
